프로젝트 소개

Docs Research Copilot은 문서 내용을 입력하면 AI가 핵심 내용을 한국어로 요약해주는 문서 요약 서비스입니다.
백엔드는 FastAPI와 Ollama를 사용해 요약 API를 제공하고, 프론트엔드는 Next.js + TypeScript로 구현해 사용자가 문서를 입력하고 결과를 확인할 수 있습니다.

현재 버전에서는 다음 기능을 지원합니다.

1. 문서 텍스트 입력

2. Ollama 기반 한국어 요약 생성

3. 입력값 검증 및 에러 메시지 처리

4. 요약 처리 단계 표시

5. 프론트엔드와 백엔드 연동

이 프로젝트는 단순한 요약 결과 제공을 넘어, LLM 기반 애플리케이션의 기본 구조를 직접 설계하고 구현하는 것을 목표로 진행될 예정입니다.
